## Set up
1. Clone project: `git clone https://github.com/uynvk/InternMiniProject.git`
2. Set up database:
    1. `cd InternMiniProject`
    2. `sudo docker compose up`
3. Run server
    1. `cd InternMiniProject/InternMiniProject`
    2. `python manage.py runserver`
